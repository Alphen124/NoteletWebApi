# Models

Data structures that represent entities in the system.

## 📋 Files

- **`user.go`** - User model (accounts, roles, permissions)
- **`device.go`** - Device model (rental items, metadata, status)

## Database Tables

- `AppUser` - User accounts
- `Device` - Rental devices
- `Review` - Device reviews and ratings
- `RentalRequest` - Rental agreements and history

See `migrations/` folder for detailed schema.
